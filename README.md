# 🍔 SmartBite – Business Analyst Portfolio Project

## 📌 Project Overview
SmartBite is a Food Ordering Application designed to streamline online food ordering, payment processing, delivery tracking, and customer management.

This project was created as a complete Business Analyst portfolio project covering end-to-end BA documentation and artifacts.

---

# 📂 Project Modules

## 1️⃣ BRD (Business Requirements Document)
Contains:
- Project overview
- Business objectives
- Scope
- Stakeholders
- Functional requirements
- Non-functional requirements

---

## 2️⃣ FRD (Functional Requirements Document)
Includes:
- Detailed system functionalities
- User actions
- Business rules
- Functional workflows

---

## 3️⃣ User Stories
Agile-style user stories with:
- User role
- Goals
- Acceptance criteria

Example:
> As a customer, I want to place food orders online so that I can receive food quickly.

---

## 4️⃣ Use Cases
Detailed use case documentation including:
- Actors
- Preconditions
- Main flow
- Alternate flow
- Postconditions

---

## 5️⃣ Wireframes
Designed using Figma.

Includes:
- Login Screen
- Signup Screen
- Home Page
- Food Menu Screen
- Checkout Screen
- Payment Screen
- Order Confirmation Screen

---

## 6️⃣ Process Flows
Created using Draw.io.

Includes:
- Login Flow
- Place Order Flow
- Payment Flow
- Refund Process Flow

---

## 7️⃣ Test Cases
Manual test cases for:
- Login
- Signup
- Order placement
- Payment validation
- Refund processing

---

## 8️⃣ RTM (Requirement Traceability Matrix)
Tracks:
- Requirements
- Test cases
- Status mapping

---

## 9️⃣ Sample Data
Contains:
- Customer data
- Order data
- Payment data
- Delivery data

Created using Excel.
